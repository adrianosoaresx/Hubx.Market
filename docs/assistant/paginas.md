# Páginas institucionais

## Para que servem páginas institucionais

Páginas institucionais ajudam a loja a explicar políticas, contato, termos, privacidade, trocas e informações sobre a marca.

Elas pertencem a uma loja específica e são isoladas por tenant.

## Como criar uma página

1. Acesse `/ops/pages/`.
2. Crie uma nova página.
3. Preencha título, slug, conteúdo e SEO básico.
4. Publique quando estiver pronta.

## Diferença entre rascunho e publicado

Página em rascunho não aparece na storefront.

Página publicada pode ser acessada publicamente em `/pages/<slug>/` dentro da loja.

## Cuidados com slug

O slug identifica a URL da página dentro da loja.

Use slugs curtos e claros, como `trocas-e-devolucoes`, `politica-de-privacidade` ou `contato`.

## Boas práticas

- Mantenha políticas objetivas.
- Revise textos antes de publicar.
- Use SEO title e description quando fizer sentido.
- Não coloque dados sensíveis, tokens ou credenciais em páginas públicas.
