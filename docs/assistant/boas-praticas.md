# Boas práticas operacionais

## Antes de publicar a loja

- Configure marca, logo, cor e hero.
- Cadastre produtos com imagens reais.
- Confira preço e estoque por variante.
- Publique páginas institucionais essenciais.
- Revise frete, checkout e pagamentos.

## Rotina diária

- Verifique pedidos pendentes.
- Acompanhe pagamentos confirmados ou falhos.
- Revise estoque baixo.
- Veja clientes que precisam de acompanhamento.
- Responda avaliações quando houver fluxo de moderação.

## Rotina semanal

- Revise produtos sem imagem ou com descrição fraca.
- Atualize páginas institucionais quando necessário.
- Confira cupons ativos.
- Verifique administradores ativos.
- Analise sinais de conversão quando disponíveis.

## O que evitar

- Apagar produto para esconder da loja.
- Tratar pagamento pendente como pago.
- Misturar comprador com admin.
- Usar dados de outra loja.
- Colar API keys, tokens ou senhas em campos livres.
