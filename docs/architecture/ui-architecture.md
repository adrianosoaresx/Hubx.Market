# UI Architecture

## Stack oficial
- Django Templates
- HTMX
- Alpine.js
- Tailwind CSS

## Objetivos
- alta produtividade
- interface consistente
- reuso de componentes
- comportamento previsível
- facilidade para atuação do Codex

## Estrutura
- `ui/templates/layouts/`
- `ui/templates/shared/components/`
- `ui/templates/shared/forms/`
- `ui/templates/shared/partials/`
- `ui/templates/patterns/`
- `ui/templates/<modulo>/`

## Governança
Toda UI deve seguir os padrões em `docs/ui/`.
