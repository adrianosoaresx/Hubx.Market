# Page Templates

## Objetivo
Definir esqueletos padrão de página.

## List page
- page header
- toolbar de ações
- filtros
- tabela ou grid
- paginação
- empty state

## Detail page
- page header
- cards de resumo
- seções
- histórico
- ações laterais quando necessário

## Form page
- page header
- descrição curta
- formulário
- ações finais

## Dashboard page
- header
- stat cards
- listas/tabelas resumidas
- blocos de atividade

## Checkout page
- etapas visuais claras
- resumo do pedido
- formulário de endereço
- frete
- pagamento
- confirmação
