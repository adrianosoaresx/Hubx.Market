# Component Library

## Objetivo
Definir os componentes reutilizáveis oficiais do Hubx Market.

## Botão
### Variantes
- primary
- secondary
- ghost
- danger
- success
- link

### Tamanhos
- sm
- md
- lg

### Estados
- default
- hover
- focus
- disabled
- loading

## Card
### Variantes
- default
- interactive
- stat
- warning
- danger

## Badge
Usado para status:
- active
- inactive
- pending
- paid
- canceled
- shipped
- delivered

## Alert
Tipos:
- success
- info
- warning
- error

## Input
Tipos:
- text
- email
- password
- number
- search

Estados:
- default
- focus
- disabled
- invalid
- success

## Select
Mesmo padrão visual e altura do input.

## Textarea
Mesmo padrão estrutural do input.

## Checkbox / Radio / Switch
Devem seguir alinhamento consistente com label e texto auxiliar.

## Table
Padronizar:
- cabeçalho
- linhas
- coluna de ações
- badges de status
- empty state

## Modal
Partes:
- header
- body
- footer
- ação primária
- ação secundária
