# Tenant Onboarding

## Fluxo oficial
1. visitante acessa Hubx Market
2. preenche dados da loja
3. escolhe subdomínio
4. informa nome do responsável
5. informa e-mail e senha
6. sistema cria Tenant + OwnerUser
7. cria configuração inicial da loja
8. redireciona para painel

## Resultado
- tenant criado
- owner vinculado
- loja pronta para configuração inicial
